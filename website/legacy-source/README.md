# 原站内容归档（nft.cup.edu.cn）

从 [中国石油大学(北京)区块链金融研究中心](https://nft.cup.edu.cn/) 抓取的静态归档，供中葡气候与能源联合研究院新站迁移使用。

> **说明**：该域名站点主体为「区块链金融研究中心」，但其中包含中葡气候与能源联合研究院揭牌等新闻与图片，故一并归档。

## 目录结构

| 路径 | 说明 |
|------|------|
| `html/` | 40 个页面的原始 HTML |
| `images/` | 新闻正文引用的图片（29 张） |
| `manifest.json` | 全站页面索引 + 抽取的正文/标题/日期 |
| `scraped-news.json` | 仅新闻/公告条目的结构化摘要 |

## 与中葡研究院直接相关

- **新闻**：`html/news_23.html` — 中葡气候与能源联合研究院在里斯本揭牌（2024-10-22）
- **图片**：`news23_1.jpg`、`news23_2.jpg`（已复制至 `website/public/images/`）

## 重新抓取

在 `website/` 目录下可运行（需网络）：

```bash
python3 scripts/scrape-legacy-site.py
```
