2026 论坛议程 — 如何提供正确文稿
==================================

当前议程在：data/forum-2026.json

您可以用以下任一方式把「无错字」的会议文件给我，我会按原文更新网站：

方式 A（推荐）
  把 Word / PDF / 图片海报 发到对话里，或上传到仓库任意目录后告诉我路径。

方式 B
  直接发纯文字（从海报或日程表复制粘贴），注明中英文是否要分开。

方式 C（自己会改文件）
  用记事本打开 data/forum-2026.json
  只改 "zh" -> "schedule" 里的文字，保存后运行：
    python3 build.py

注意：请提供最终定稿文字，我会原样写入，不再改写。
