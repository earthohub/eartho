首页会议 / 活动图片 — 放这里即可
================================

1. 把您的照片复制到本文件夹（本目录）

2. 建议文件名（按显示顺序）：
   01.jpg
   02.jpg
   03.jpg
   …
   也可用：photo-1.jpg、photo-2.png 等

3. 支持格式：.jpg  .jpeg  .png  .webp

4. 放好后在项目里运行：
   cd cp-jri-static
   python3 build.py

5. 首页会自动出现「活动影像 / Gallery」区块展示这些图。

可选：若需固定顺序或说明文字，编辑 data/home-gallery.json
（见该文件内注释）。
